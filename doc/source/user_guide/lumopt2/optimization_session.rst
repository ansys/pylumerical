Setting up the optimization session
====================================

.. note::
   This article is under construction.

This article covers combining lumopt2 components into a complete optimization workflow, including:

- Creating a ``Project`` from setup, parametrization, and figure of merit
- Configuring the ``ScipyOptimizer``
- Setting up the ``Optimization`` class with callbacks
- Running the optimization with ``Optimization.run()``
- Understanding ``OptimizationResult`` and post-optimization analysis
