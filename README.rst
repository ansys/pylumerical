About
=====

PyLumerical is a Python library that offers a powerful and user-friendly API
for interacting with `Ansys Lumerical`_. It is designed to enhance workflows by
providing a comprehensive set of tools and functionalities.

With PyLumerical, developers can seamlessly integrate Lumerical into their
Python applications. It enables efficient automation, supports the creation of
custom applications, and integrates well with other popular Python libraries.

Installation
============

Ensure you have all the necessary prerequisites. Then, refer to the
installation guidelines for detailed instructions on how to install PyLumerical
in your system.
